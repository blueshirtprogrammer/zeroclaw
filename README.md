## What is Agent ZERO V 2.2?

Agent ZERO V 2.2 is an AI-powered assistant that can:
- Control your Linux environment
- Install software and manage files
- Execute commands and write code
- Use the internet for research
- Run browser automation
- Process voice input/output
- Much more...

It's like having a brilliant intern who can actually use your computer.

### Docker (Recommended)
```bash
docker pull blueshirtprogrammer/agent-zero-x
docker run -p 50001:80 blueshirtprogrammer/agent-zero-x
```

### From Source
```bash
git clone https://github.com/blueshirtprogrammer/agent-zero-x.git
cd agent-zero-x
./run_ui.py
```
